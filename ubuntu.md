# Ubuntu

1. Mettre à jour et redemarrer le système d'exploitation

2. Faire un `sudo snap refresh` et redemarrer

3. Installer curl : sudo apt install curl

4. Installer Git : https://git-scm.com/install/linux <br>
   `git config --global user.name "John Doe"` <br>
   `git config --global user.email johndoe@example.com`

5. Installer MariaDB : https://www.digitalocean.com/community/tutorials/how-to-install-mariadb-on-debian-11
 
6. Installer PHP et Composer via la CLI de Laravel : https://laravel.com/docs/13.x/installation

7. Installer NodeJS : https://nodejs.org/en/download

8. Installer "Extensions" via Snap

9. Installer DashToPanel via un navigateur
